# Otus  iOS professional home work #2

## Build instructions

1. Place `.env` file in the root directory of the project.
Example of .env file:
```
# newsapi.ai key
newsApiKey=xxx-xxx-xxx
fetchArticlesCount=20
```
2. Enable `BuildEnvFilePlugin` and `OpenAPIGenerator` plugins.
